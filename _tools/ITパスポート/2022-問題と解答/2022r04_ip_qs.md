参考：擬似言語の記述形式 (ITパスポート試験用) 

**記述形式**

* `○手続名又は関数名`: 手続又は関数を宣言する。
* `型名: 変数名`: 変数を宣言する。
* `/* 注釈 */` または `// 注釈`: 注釈を記述する。
* `変数名 ← 式`: 変数に式の値を代入する。
* `手続名又は関数名(引数, ...)`: 手続又は関数を呼び出し、引数を受け渡す。
* `if (条件式1) ... elseif (条件式2) ... else ... endif`: 選択処理を示す。
* `while (条件式) ... endwhile`: 前判定繰返し処理を示す。
* `do ... while (条件式)`: 後判定繰返し処理を示す。
* `for (制御記述) ... endfor`: 繰返し処理を示す。

**演算子と優先順位**
優先度 高 → 低

1. `()`
2. `not`, `+`, `-` (単項)
3. `*`, `/`, `mod` (乗除)
4. `+`, `-` (加減)
5. `>`, `<`, `≥`, `≤`, `=`, `≠` (関係)
6. `and` (論理積)
7. `or` (論理和)

**論理型の定数**
`true`, `false`

**配列**

* 一次元配列: `[ ]` でアクセス。例 `exampleArray[4]`
* 二次元配列: `[行, 列]` でアクセス。例 `exampleArray[2, 5]`
