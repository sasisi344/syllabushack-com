# ITパスポートはCBT体験できるアプリがある

## アプリをDLできるURL
https://www3.jitec.ipa.go.jp/JitesCbt/html/guidance/trial_examapp.html

本番同様のアプリを使えるので、他のCBT試験の対策にも使える。
アプリを使ってどのような操作をするのか、に慣れるには最適。過去問で試せるから勉強にも使える。

なので、IPAをのIT系資格試験対策には最適かもしれない。

## ITパスポート自体が全体の基礎的な部分がある
ITパスポートで出題されるのは、何もPC関連の知識だけじゃない。
2進数だったりBASICレベルのプログラミング言語も出るし、ストラテジストの組織管理なりチームのタスク管理能力まで多岐にわたる。
出題範囲こそ広いけど、いろいろなビジネス本を読んでいつつ、ふだんからPCに触れて操作以外に知識を蓄えているなら、それだけで6割以上は取れると思う。
100問中の8割以上が理想だから、ノー勉強で挑むのはリスクがある。
まずは過去問をCBT体験でやってみて、何点取れるかで試験対策をするのが妥当だろう。

だからITパスポートに関しては、試験テキストはあってもいいけど必ず必要ってほどじゃない。
ここ数年で大きく変わったのは、生成AIとLLM知識くらい。
毎年シラバスは大きく変わるわけじゃないので、昨年との違いを明確にすれば、毎年テキストなしで試験対策をすることができる。
