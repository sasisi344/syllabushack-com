# syllabus-index-data
IPA（独立行政法人情報処理推進機構）が実施するIT系国家資格試験は、スキルレベル（難易度）や専門分野に応じて以下の通り分類されています。

### レベル1（ITを利活用する者）
*   **ITパスポート試験（IP）**：ITに関する基礎的な知識を証明する試験です,。

### レベル2（情報処理技術者）
*   **情報セキュリティマネジメント試験（SG）**：情報セキュリティマネジメントの計画・運用・評価・改善を通して組織の情報セキュリティ確保に貢献し、脅威から継続的に組織を守るための基本的なスキルを認定する試験です,。
*   **基本情報技術者試験（FE）**：ITエンジニアの登竜門とされ、ITを活用したサービス、製品、システム及びソフトウェアを作る人材に必要な基本的知識・技能を認定する試験です,。

### レベル3（応用情報技術者）
*   **応用情報技術者試験（AP）**：高度IT人材となるために必要な応用的知識・技能をもち、高度IT人材としての方向性を確立した者を対象とした試験です,。

### レベル4（高度情報処理技術者）
高度な専門的知識と技能を要する試験区分で、専門分野ごとに以下のように分かれています。

**【高度試験】**
*   **ITストラテジスト試験（ST）**：経営戦略に基づいてIT戦略を策定し、ITを活用した事業革新、業務改革、革新的な製品・サービスの創出を企画・推進する人材を対象とします,。
*   **システムアーキテクト試験（SA）**：システム開発の要件定義、アーキテクチャの設計を行い、システム開発を主導する人材を対象とします,。
*   **プロジェクトマネージャ試験（PM）**：システム開発プロジェクトの計画立案、予算・工程・品質管理などを行い、プロジェクトを成功に導く責任者を対象とします,。
*   **ネットワークスペシャリスト試験（NW）**：ネットワークシステムの企画・要件定義・開発・運用・保守に関わる専門技術者を対象とします,。
*   **データベーススペシャリスト試験（DB）**：データベースシステムの企画・要件定義・開発・運用・保守に関わる専門技術者を対象とします,。
*   **エンベデッドシステムスペシャリスト試験（ES）**：IoTを含む組込みシステムの企画・要件定義・開発・運用・保守に関わる専門技術者を対象とします,。
*   **ITサービスマネージャ試験（SM）**：情報システムの安全性と信頼性を確保し、サービスの提供・維持・改善を行う運用管理者を対象とします,。
*   **システム監査技術者試験（AU）**：情報システムのリスクやコントロールを独立した立場で点検・評価・検証し、改善提言を行う監査人を対象とします,。

**【情報処理安全確保支援士試験】**
*   **情報処理安全確保支援士試験（SC）**：サイバーセキュリティに関する高度な知識・技能をもち、セキュアな情報システムの企画・設計・開発・運用を支援する専門家を認定する試験です。合格し登録することで国家資格「情報処理安全確保支援士（登録セキスペ）」の資格保持者となります,。
