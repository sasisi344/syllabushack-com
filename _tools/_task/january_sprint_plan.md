# 2026年1月 残タスク＆スプリント計画：Syllabus Hack Pivot

## 🎯 戦略転換（Pivot）
**「魚（過去問）」を与えるのではなく、「釣具（生成AIハックツール）」を与えるサイトへ。**

*   **Core Value:** どんな資格試験も、公開されたシラバスさえあれば、AIを使って最短で攻略できる「メソッド」と「ツール」を提供する。
*   **First Showcase:** IPA資格（基本情報・応用情報）を題材に、このメソッドが有効であることを実証する。

## 📅 1月残り（〜1/31）のゴール
2月からの本格始動に向け、**「ハックメソッドの確立」**と**「プロトタイプ記事の公開」**を完了させる。

*   [ ] **シラバス構造化プロンプトの開発**
