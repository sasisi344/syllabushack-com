# Agent Teacher

## 「教えてもらう」から「使い倒す」へ

従来の学習は、先生や参考書が「教えてくれる」のを待つ受動的なものでした。
Agent Teacher は、生成AI（ChatGPT, Gemini, Claude）を**自分専属の家庭教師**として定義し、能動的に使い倒す学習スタイルです。

### できること

*   **24時間365日の質問対応**:
    *   「今の説明、もっと簡単に言って？」
    *   「この用語、小学生にもわかるように例えて」
    *   なんていうワガママな質問にも、AIは無限に付き合ってくれます。
*   **パーソナライズされたカリキュラム**:
    *   あなたの苦手分野を分析し、「今日やるべき3問」を提示します。
*   **メンタルコーチング**:
    *   モチベーションが落ちた時、学習が進まない時、AIが励まし、ペース配分を再調整してくれます。

### 始めるには

まずは [Method カテゴリ](/blog/method/) の記事で紹介している「プロンプト」をコピーして、お使いのAIに投げかけてみてください。
その瞬間から、あなたの学習体験は劇的に変化します。
