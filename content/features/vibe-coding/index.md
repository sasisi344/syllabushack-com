# Vibe Coding

## Generate, Don't Write.

Vibe Coding（バイブ・コーディング）とは、本来は「AIを使って感覚的・直感的にコードを書く」というプログラミング用語ですが、Syllabus Hackではこれを**学習コンテンツの生成**に応用しています。

### 学習における Vibe Coding

*   **自分専用問題集の生成**:
    *   過去問は有限ですが、AIが生成する類題は**無限**です。
    *   Pythonスクリプトによる「トレース表」の自動生成や、暗記カードの作成など、面倒な作業はすべてAIに任せましょう。
*   **データの可視化**:
    *   学習進捗や苦手分野を、AIにグラフ化させることも可能です。

### Philosophy

「人間は思考（Vibe）に集中し、実作業（Coding/Writing）はAIが行う」
これがSyllabus Hackが提案する、新時代の知的生産スタイルです。
