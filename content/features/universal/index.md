# Universal Syllabus

## 資格試験の「OS」をハックする

Syllabus Hack のメソッドは、特定の試験（例えば基本情報技術者）だけに特化したものではありません。
**シラバス（試験範囲）が存在するすべての試験**に適用可能です。

### なぜ Universal なのか？

すべての資格試験は、以下の共通構造を持っています。

1.  **シラバス**: 何を問うかが定義されている仕様書。
2.  **過去問**: シラバスに基づいたサンプリングデータ。
3.  **評価基準**: 合格ラインという閾値。

この構造さえあれば、AIはそこから逆算して最適な学習プランと教材を生成できます。

### 対応予定ジャンル

*   **IT系**:
    *   ITパスポート、基本情報、応用情報、高度試験
    *   AWS認定、G検定（AI）
*   **ビジネス系**:
    *   簿記、FP（ファイナンシャルプランナー）
    *   中小企業診断士
*   **語学**:
    *   TOEIC、英検

「この資格のハック方法が知りたい」というリクエストがあれば、ぜひSNS等でお知らせください。
シラバスを解析し、攻略プロンプトを開発します。
